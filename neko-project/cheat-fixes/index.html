Loader closing while loading without opening game

Solution

  1. Generate a new loader and put it in its own folder.
  2. Uninstall and reinstall NVIDIA App.
  3. Uninstall and reinstall Overwolf.
  4. Reinstall Visual C++ Redistributables.
  5. Reinstall DirectX End User Runtimes.
  6. Restart your PC.

Error: 0xC0000100

Solution

Try using a VPN or simply try loading again.

Solution

Solution

Reinject a few times. If the issue persists, you may need to reinstall Windows.

NVIDIA: Failed Overlay

Solution

Open NVIDIA App, disable the overlay, and then re-enable it.

Radeon: No Menu

Solution

Open Overwolf, go to "Overlay & Hotkeys", and enable the Rust toggle.

Menu frozen / Not interactable

Solution

1. Locate Rust Executable

C:\Program Files (x86)\Steam\steamapps\common\Rust

2. Adjust Properties

Right-click Rust executable > Properties > Compatibility Tab

  • Uncheck "Run this program as an administrator"
  • Click Change high DPI settings
  • Disable "Use this setting to fix scaling problems"
  • Disable "Override high DPI scaling behavior"

3. Finalize

Click Apply, restart Steam, and launch Rust.

Error: Try Loading a VPN

Solution

Install a free VPN, connect to it, then try injecting the cheat again. You can disconnect after the cheat has loaded.
