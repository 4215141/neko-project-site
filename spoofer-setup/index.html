Troubleshooting

Initialization Failure

Open cmd.exe as admin and run the following command, then restart your PC:

bcdedit /set hypervisorlaunchtype off

Disable TPM in BIOS

  • Restart PC and enter BIOS setup (usually F2, DEL, or F10).
  • Navigate to Security or Advanced tab.
  • Locate TPM (Trusted Platform Module) option.
  • Set to Disabled.
